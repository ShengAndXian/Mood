# Mood v1.0
![](http://olrdynjoh.bkt.clouddn.com/icon-83.5@2x.png)

## 一、功能介绍

### 1、每日心情

每个用户每天都能且仅能添加一条心情记录。

![](http://olrdynjoh.bkt.clouddn.com/IMG_208412.PNG)

### 2、心情卡片

用户的心情记录将以卡片的形式显示在app的主界面。



### 3、心情数据

用户所产生的心情数据将会以图表的形式呈现。



### 4、用户定制

用户可以通过设置界面对各个界面显示的内容进行定制。



### 二、安装

下载整个项目文件后，通过pod安装一下第三方库，然后在XCode中运行即可。
